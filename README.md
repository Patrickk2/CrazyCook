# CrazyCook 🍳

**CrazyCook** est un site de vente de cuisine.
---

## 🚀 Installation rapide
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/<votre-nom-utilisateur>/CrazyCook.git
   cd CrazyCook
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Lancez le serveur de dev :
   ```bash
   npm start
   ```
   Accédez à `http://localhost:3000`.

---
